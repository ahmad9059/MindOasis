# **App Name**: Mind Oasis

## Core Features:

- Therapist Listing: Display a searchable and filterable list of therapists.
- Profile Search: Implement a full-text search across therapist profiles including name, expertise, education, and about sections.
- Detailed View: Display detailed information about each therapist, including their biography, expertise, consultation mode, and contact information.
- Filtering: Implement filtering based on city, experience, gender, fee range, and consultation mode.
- Smart Insights Generator: Utilize a tool that employs a Large language model to summarize therapist qualifications, experience, and specializations, assisting the LLM in producing comprehensive and insightful profile summaries.
- Landing Page: Design an engaging landing page that introduces the application and its features.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to create a sense of tranquility and trust.
- Background color: Light gray (#F5F5F5) for a clean, modern, and neutral backdrop.
- Accent color: Soft orange (#FFAB91) to highlight key interactive elements and calls to action.
- Body and headline font: 'Inter', a sans-serif font known for its readability and modern design.
- Use clear, professional icons to represent filter options, contact information, and other key UI elements. Icons from 'lucide-react' are suggested.
- Employ a responsive two-column layout, with filters on the left and therapist listings on the right. Use a card-based design for therapist profiles to maintain consistency.
- Incorporate subtle transitions and loading states to enhance user experience, for instance when displaying or filtering the results.
